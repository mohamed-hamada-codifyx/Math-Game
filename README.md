# ➕ ➖ ✖️ ➗ Math Game (C++)

A simple and interactive math quiz game built with C++.

---

### 🕹️ How to Play
* **Step 1:** Select the number of questions you want to solve.
* **Step 2:** Choose the difficulty level (Easy, Medium, Hard, or Mixed).
* **Step 3:** Select the operation type (Add, Sub, Mul, Div, or Mix).
* **Step 4:** Answer the questions and see your final score at the end.

---

### 📥 How to Run the Game
1. **Download:** Save the `main.cpp` file to your computer.
2. **Compile:** Use a C++ compiler to build the project:
   ```bash
   g++ main.cpp -o MattGame
